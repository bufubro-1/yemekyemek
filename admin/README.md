# YemekYemek Yönetim Paneli

React ve Vite tabanlı yönetim uygulaması.

## Çalıştırma

```bash
npm install
npm run dev
```

Uygulama `http://localhost:5173` adresinde çalışır ve varsayılan olarak
`http://localhost:3000/v1` API adresini kullanır.

API adresini değiştirmek için:

```bash
VITE_API_URL=http://localhost:3000/v1 npm run dev
```

## Kontroller

```bash
npm run lint
npm run build
```
